# Partnership

A simple two-person daily mindfulness practice app.

## Run locally

From the project folder:

```bash
cd /workspace/Partnership
python3 -m http.server 4173
```

Then open:

- http://localhost:4173

## Notes

- This app is static (HTML/CSS/JS), so you must serve it with a local web server (not just double-clicking the file) for all browser features to work consistently.
- Firebase is optional right now; if you don't configure it, the app still works in local/offline mode using localStorage.
